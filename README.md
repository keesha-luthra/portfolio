# Keesha Luthra - Personal Portfolio

This is my personal portfolio website built with [Next.js](https://nextjs.org/), TypeScript, and custom CSS. 
It showcases my personal projects, research, beliefs, and professional journey.

## Features
- **Dynamic & Modern Design**: Clean and interactive layout with attention to detail.
- **Custom Cursor**: Interactive custom cursor that changes states.
- **Projects Showcase**: Detailed project cards that route to dedicated project pages.
- **Fully Responsive**: Optimized for both mobile and desktop screens.

## Getting Started

First, install the dependencies if you haven't already:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the live portfolio.

## Deployment

This project is optimized for direct deployment on [Vercel](https://vercel.com/). Simply connect your GitHub repository to Vercel and it will automatically build and deploy your site.
